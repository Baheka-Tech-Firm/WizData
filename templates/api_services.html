Available API Endpoints

Symbols

Get available symbols for different asset types

Endpoint: /api/symbols

Method: GET

Parameters:

  • asset_type - The asset type (jse, crypto, forex)

Example:

/api/symbols?asset_type=crypto

Prices

Get price data for specific symbols

Endpoint: /api/prices

Method: GET

Parameters:

  • asset_type - The asset type (jse, crypto, forex)
  • symbol - The ticker symbol (e.g., "SOL", "BTC")
  • interval - The time interval (daily, weekly, monthly)
  • start_date - Start date in YYYY-MM-DD format
  • end_date - End date in YYYY-MM-DD format

Example:

/api/prices?asset_type=jse&symbol=SOL&interval=daily&start_date=2023-01-01&end_date=2023-01-31

API Documentation

Authentication

All API requests require an API key that should be included in the header of your request.

GET /api/symbols?asset_type=crypto HTTP/1.1
Host: api.wizdata.com
Authorization: Bearer YOUR_API_KEY
Accept: application/json
Rate Limiting

Rate limits are applied based on your subscription plan:

  • Basic: 100 requests per day
  • Standard: 1,000 requests per day
  • Premium: 10,000 requests per day
  • Enterprise: Custom limits
Error Handling

The API uses standard HTTP status codes and returns detailed error messages in the following format:

{
  "status": "error",
  "message": "Invalid asset type: unknown",
  "code": "INVALID_PARAMETER"
}
Code Examples
Python
import requests

url = "https://api.wizdata.com/api/prices"
params = {
    "asset_type": "jse",
    "symbol": "SOL",
    "interval": "daily",
    "start_date": "2023-01-01",
    "end_date": "2023-01-31"
}
headers = {"Authorization": "Bearer YOUR_API_KEY"}

response = requests.get(url, params=params, headers=headers)
data = response.json()

print(f"Retrieved {data['count']} price records")
JavaScript
const fetchPrices = async () => {
  const url = new URL('https://api.wizdata.com/api/prices');
  url.searchParams.append('asset_type', 'crypto');
  url.searchParams.append('symbol', 'BTC');
  url.searchParams.append('interval', 'daily');
  
  const response = await fetch(url, {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  });
  
  const data = await response.json();
  console.log(`Retrieved ${data.count} price records`);
};
