Financial Data API

Get Stock Data

GET /api/integrated/stock/AAPL

# Response
{
  "symbol": "AAPL",
  "company_name": "Apple Inc.",
  "stock_data": {
    "latest_price": 178.72,
    "change_30d": 5.2,
    "change_90d": -2.1,
    "change_1y": 15.3
  },
  "time_series": [...]
}

Get Forex Data

GET /api/integrated/forex?from_currency=USD&to_currency=EUR

# Response
{
  "from_currency": "USD",
  "to_currency": "EUR",
  "data": {
    "exchange_rate": 0.9234,
    "last_refreshed": "2025-04-04 08:15:00"
  }
}

ESG Data API

Get Country ESG Data

# Response

  "country_code": "ZA",
  "country_name": "South Africa",
  "esg_scores": {
    "environmental": 68.3,
    "social": 62.8,
    "governance": 71.5,
    "infrastructure": 65.7,
    "overall": 67.1
  }
}
